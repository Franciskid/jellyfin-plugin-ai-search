AI Search & Recommendation

Adds an AI search bar to the web client that recommends movies from your own library. Platform mode routes through a self-hosted AI platform that keeps a semantic index of your library; Direct mode calls any OpenAI-compatible endpoint (OpenAI, Ollama, OpenRouter, …) itself.

Semantic index

Builds a local embedding index of your movies so search retrieves by meaning instead of sending a slice of the catalog. The index refreshes nightly (Scheduled Tasks → "Build AI Search index") and only re-embeds what changed.

Enables the "TV Shows" search scope by embedding your series and every episode too. Episode counts can be large, so the first build after enabling this may run for a while. Rebuild the index after changing it.
